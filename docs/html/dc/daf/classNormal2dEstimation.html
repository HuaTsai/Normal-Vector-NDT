Detailed Description

Normal2dEstimation estimate local straight line at each points using it neighbour. Contrary to native normal estimation on PCL lib, it works only on 2D points cloud.

Note
This class use 3D points cloud but ignore all z components. So make sure you have all your points on z=0 plan.
